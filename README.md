**FEATURES**

1. Selection of 3 different quizes, each with 10 questions
2. Timer per question, that will push no answer forward if an answer is not commited to
3. Answer selection is highlighted on click to aid users decisions
4. Question number is shown to user
5. Score is shown to user as quiz is being conducted and upon conclusion
6. List of all questions answers correctly and incorrectly are shown on final card after attempting quiz


**TECH STACK**

- Front End : JavaScript, HTML, CSS


**SET UP INSTRUCTIONS**

Run the following link to open quiz:
https://group2quizgame.netlify.app/


**USAGE**

1. Upon opening app, select quiz you would like to participate in
2. Select the answer you find to be correct and select "Submit" to confirm your choice
3. Continue to answer the questions within the time limit, and select "Submit" without any answers highlighted for any questions you are unsure of to ignore the question
4. Upon conpletion, you will see a score card with you total out of 10, as well as a list of questions you answered correctly and incorrectly
